Plum FAQ Chatbot

This is a demonstration of an embeddable AI chatbot widget. Click the blue circle in the bottom-right corner to start chatting!

Features

  • Collapsible chat interface
  • Smooth animations and transitions
  • Ready for API integration
  • Mobile-responsive design
  • Perfect for iframe embedding

How to Use

  1. 1 Click the floating blue button to open the chat
  2. 2 Type your message in the input field
  3. 3 Press Enter or click Send to send your message
  4. 4 Click the X button to close the chat

Integration Guide

To embed this chatbot in your main application via iframe:

<iframe src="your-chatbot-url" style="position: fixed; bottom: 0; right: 0; width: 400px; height: 600px; border: none; z-index: 1000;" title="AI Chatbot"> </iframe>

The widget is designed to be fully self-contained and will position itself correctly within the iframe.
